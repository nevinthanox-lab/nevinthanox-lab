## 💡 Design Principles I Follow

- **Fail-closed by default** — every agent errs toward escalation, never silent failure
- **Schema-validated outputs** — strict JSON contracts, no hallucinated structure
- **Human-in-the-loop** — high-risk decisions always wait for approval
- **Classical ML → Agent handoff** — models score and explain, agents act
