I design AI systems the way a production engineer would, not the way a notebook demo would: **classical ML models score and explain**, **guardrailed LLM agents act on those scores**, and **nothing high-risk executes without human sign-off**. Six end-to-end systems shipped so far — every one **fail-closed**, **schema-validated**, and **fully traceable**.

## 🛡️ Engineering Principles

<div align="center">

| Principle | What it means in practice |
|:---|:---|
| **Fail-closed by default** | Every agent errs toward escalation — never a silent failure |
| **Schema-validated outputs** | Strict JSON contracts, zero hallucinated structure |
| **Human-in-the-loop** | High-risk decisions always wait for explicit approval |
| **Classical ML → Agent handoff** | Models score and explain; agents execute the decision |
